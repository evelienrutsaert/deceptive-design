# Deceptive Patterns — Theorielijst

Een interactief naslagwerk dat alle gangbare **deceptive design patterns** koppelt aan de **cognitieve principes** die ze exploiteren. Gebouwd als lesmateriaal voor studenten en ontwerpers die willen begrijpen hoe misleidende interfaces werken — en waarom ze effectief zijn.

## Overzicht

De theorielijst bevat **16 deceptive patterns** verdeeld over drie categorieën, elk gekoppeld aan **21 unieke psychologische principes** uit de gedragswetenschap. Klik op een principe om een uitgebreide uitleg te lezen, inclusief de achterliggende onderzoeker(s) en in welke patronen het voorkomt.

### Categorieën

| Kleur | Categorie | Omschrijving |
|---|---|---|
| 🔴 | **Misleiden** | Informatie verborgen, vervormd of gefaseerd onthuld |
| 🔵 | **Manipuleren** | Feiten kloppen, maar de presentatie stuurt via biases en emotie |
| ⚫ | **Obstrueren** | Makkelijk in, bewust moeilijk uit |

### Patronen

**Misleiden**
- Hidden costs / Drip pricing
- Sneaking
- Trick wording
- Visual interference
- Disguised ads
- Comparison prevention

**Manipuleren**
- Fake scarcity
- Fake urgency
- Fake social proof
- Preselection
- Confirmshaming
- Nagging

**Obstrueren**
- Hard to cancel / Roach motel
- Hidden subscription / Forced continuity
- Obstruction
- Forced action

## Features

- **Filter op categorie** — toon alleen patronen uit Misleiden, Manipuleren of Obstrueren
- **Zoekfunctie** — doorzoek patronen en principes op naam of trefwoord
- **Interactieve principes** — klik op een blauw principe-label voor een modal met uitleg, bronvermelding en overzicht van gerelateerde patronen
- **Responsive design** — werkt op desktop en mobiel
- **Geen dependencies** — volledig standalone HTML-bestand

## Gebruik

Open `index.html` in een browser. Er zijn geen build-stappen of dependencies nodig.

```bash
open index.html
```
